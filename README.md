# 🐟 摸鱼屏保 (Loafing Screen Saver)

这是一个基于 Python + PyQt6 编写的趣味屏保程序，专为“摸鱼”场景设计。
支持多种逼真的系统故障界面，帮助你优雅地“带薪休息”。

## ✨ 功能特点
- **多场景伪装**：支持 Windows 系统更新、蓝屏 (BSOD)、系统恢复、系统重启、FBI 警告等多种界面。
- **安全机制**：禁用 Esc 键，需通过组合键或特定操作退出，防止误触露馅。
- **高清画质**：使用矢量字符动画和系统原生字体，支持高分屏，细节逼真。

## 🚀 如何使用 (直接运行)
1. 在右侧 **[Releases]** 页面下载最新的 `.exe` 文件。
2. 双击运行，选择想要的伪装模式。
3. 点击“开始伪装”。

## 🛠️ 如何退出 (请牢记)
由于为了逼真禁用了 Esc 键，请使用以下任意一种方式退出：
- **键盘**：按下 `Ctrl` + `Alt` + `Q`
- **鼠标**：移动到屏幕 **左上角** 连续点击 3 次
- **右键**：如果开启了右键菜单，点击右键选择“恢复桌面”
