# 🐟 摸鱼屏保 (Loafing Screen Saver)

这是一个基于 Python + PyQt6 编写的趣味屏保程序，专为“摸鱼”场景设计。

## ✨ 功能特点
- **多场景伪装**：支持 Windows 系统更新、蓝屏 (BSOD)、系统恢复、FBI 警告等多种界面。
- **安全机制**：禁用 Esc 键，需通过组合键或特定操作退出，防止露馅。
- **高清画质**：使用矢量字符动画和系统原生字体，支持高分屏。

## 🚀 如何使用 (直接运行)
1. 在右侧 [Releases] 页面下载最新的 `.exe` 文件。
2. 双击运行，选择伪装模式。
3. 点击“开始伪装”。

## 🛠️ 如何退出
- **键盘**：按下 `Ctrl` + `Alt` + `Q`
- **鼠标**：点击屏幕 **左上角** 连续 3 次
- **右键**：如果开启了右键菜单，选择“恢复桌面”
